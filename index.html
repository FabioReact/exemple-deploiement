Déployer une page statique sur github

Pré-requis: Avoir un compte Github et avoir Git installé sur sa machine

  1. Initialiser le dépot git avec la commande "git init"
  2. Il faudra ensuite ajouter les fichiers à notre index - Pour cela on éxécutera la commande "git add ."
  3. Il faudra sauvegarder localement nos changements en faisant un commit (enregistrement) avec la commabde "git commit -m 'notre commentaire qui décrit le travail effectué'"
  4. Sur github, Créer un nouveau "Repository", puis suivre les étapes indiquées
  5. Une fois notre code en ligne, pour l'afficher sur un domaine github, il faudra l'aller dans l'onglet "Settings" de notre repository
  6. Scroller vers le bas de page, jusqu'à trouver l'onglet "Github Pages"
  7. Séléctioner comme source (menu déroulant) la branche "main", puis sauvegardez
  8. Une fois la page mise à jour, allez sur le lien nouvelement créer
